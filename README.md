# Kółko i Krzyżyk – dokumentacja projektu

## Informacje ogólne
Repozytorium dokumentuje proces tworzenia gry Kółko i Krzyżyk.
Projekt skupia się na planowaniu, etapach produkcji oraz organizacji pracy
z wykorzystaniem systemu kontroli wersji Git i platformy GitHub.

## Plan gry
Gra Kółko i Krzyżyk to gra dla dwóch graczy,oparta na klasycznej planszy 3x3.
Gracze wykonują ruchy naprzemiennie symbolami O i X.

## Plansza gry
Plansza gry została zaprojektowana jako siatka 3x3 bez bocznych krawędzi.
Każde pole planszy może przyjmować jeden z trzech stanów: puste, X lub O.

## Logika gry
Logika gry obejmuje obsługę ruchów graczy, sprawdzanie poprawności ruchu
oraz zmianę aktualnego gracza po każdym ruchu.  
Każda tura kończy się sprawdzeniem, czy któryś z graczy wygrał lub czy nastąpił remis.
